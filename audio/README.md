Put animal sound MP3 files in this folder.

Current filenames used by script.js:

- bison.mp3
- lion.mp3
- tiger.mp3
- african-elephant.mp3
- asian-elephant.mp3
- panda.mp3
- wolf.mp3
- kangaroo.mp3
- poison-dart-frog.mp3
- emperor-penguin.mp3
- african-savanna-elephant.mp3

To add another sound, add an `audio: "./audio/file-name.mp3"` field to that animal's detail object in `script.js`.
